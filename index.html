Looking at the summary data it seems like there are a number of outliers and misentered data. Let's clean that up a bit.

RemoveOutliers function will take the taxi dataframe, substitute negative fares with its median, and calculates its lower bound and upper bound using the IQR Rule.

In [56]:
RemoveOutliers <- function(data) {
    data$total_amount[data$total_amount <= 0] <- median(data$total_amount)
    lowerbound <- quantile(data$total_amount, 0.25) - 1.5*IQR(data$total_amount)
    upperbound <- quantile(data$total_amount, 0.75) + 1.5*IQR(data$total_amount)
    data <- dplyr::filter(data, total_amount >= lowerbound, total_amount <= upperbound)
    return(data)
}

Let's calculate a 95% confidence interval to estimate the true population mean of each borough's average total fare amount.¶

I will employ the t-statistic since I am working with the sample standard deviation.

In [60]:
ConfidenceInterval.TDist <- function(data, bname) {
    data.n <- length(data$total_amount)
    data.a <- mean(data$total_amount)
    data.d <- sd(data$total_amount)
    error <- qt(0.975, df = data.n-1)*data.d/sqrt(data.n)
    conf.interval <- data.frame(borough.name = bname, lower.bound = data.a - error, mean = data.a, upper.bound = data.a + error)
    return(conf.interval)
}

EWR.CI <- ConfidenceInterval.TDist(EWR.tripdata, "EWR")

Another interesting test we can conduct is to further split the data into two different zones (Boro zones and Yellow zones) and conduct an unpaired two-samples t-test to compare their means.¶

Boro taxis are taxis in NYC that are allowed to pick up passengers in outer boroughs(the Bronx, Brooklyn, Queens, and Staten Island) and in Manhattan above East 96th and West 110th Streets.

Yellow taxis however can pick up passengers from anywhere.

Preliminary test to check independent t-test assumptions¶

Assumption 1: Are the two samples independents? YES

Assumption 2: Are the data from each of the 2 groups follow a normal distribution? YES

Assumption 3. Do the two populations have the same variances? We'll need to conduct an F-test

In [149]:
s1 <- sample_n(BORO.tripdata, 100)
s2 <- sample_n(YELLOW.tripdata, 100)

res.ftest <- var.test(s1$total_amount, s2$total_amount)

The p-value of F-test is p = 0.03565. Its less than the significance level alpha = 0.05.

Thus, there is a significant difference between the variances of the two sets of data. Therefore, we cannot use the classic t-test which assume equality of the two variances.

Instead, we will use Welch's Two Sample t-test.

t.test(s1$total_amount, s2$total_amount, var.equal = FALSE)
