Looking at the summary data it seems like there are a number of outliers and misentered data. Let's clean that up a bit.¶

RemoveOutliers function will take the taxi dataframe, calculates its lower bound and upper bound using the IQR Rule.¶
In [9]:
RemoveOutliers <- function(data) {
    lowerbound <- quantile(data$total_amount, 0.25) - 1.5*IQR(data$total_amount)
    upperbound <- quantile(data$total_amount, 0.75) + 1.5*IQR(data$total_amount)
    data <- dplyr::filter(data, total_amount >= lowerbound, total_amount <= upperbound)
    return(data)
}

Let's finish up by calculating a 95% confidence interval to estimate the true population mean of each borough's average total fare amount. I will employ the t-test since I am working with the sample standard deviation.¶

In [74]:
ConfidenceInterval.TDist <- function(data, bname) {
    data.n <- length(data$total_amount)
    data.a <- mean(data$total_amount)
    data.d <- sd(data$total_amount)
    error <- qt(0.975, df = data.n-1)*data.d/sqrt(data.n)
    conf.interval <- data.frame(borough.name = bname, lower.bound = data.a - error, mean = data.a, upper.bound = data.a + error)
    return(conf.interval)
}

EWR.CI <- ConfidenceInterval.TDist(EWR.tripdata, "EWR")
Bronx.CI <- ConfidenceInterval.TDist(Bronx.tripdata, "Bronx")
Brooklyn.CI <- ConfidenceInterval.TDist(Brooklyn.tripdata, "Brooklyn")
Manhattan.CI <- ConfidenceInterval.TDist(Manhattan.tripdata, "Manhattan")
Queens.CI <- ConfidenceInterval.TDist(Queens.tripdata, "Queens")
Staten.CI <- ConfidenceInterval.TDist(Staten.tripdata, "Staten Island")
